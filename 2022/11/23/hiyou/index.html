成品简介

【1.0版本】

该版本由Arduino UNO主板一块儿、1.3寸OLED屏幕一块儿、DS3231时钟模块一个、无源蜂鸣器一个、面包板一块儿、按钮四个、10kΩ电阻四个、跳线杜邦线若干组成。

具有显示时间和环境温度功能

在屏幕上分别显示着当前的年、月、日、时、分、秒、环境温度（可以通过按动一次红色按钮在摄氏温度与华氏温度之间相互转换），以及一个数学公式（下文介绍其用途）。

具有闹钟功能

使用者可以通过两个按钮来分别设置闹钟时间的时和分（24小时制）。通过DS3231时钟模块返回的当前时间值与使用者设置的时间比较，若相同则开始响铃。
强力唤醒功能
为了使使用者尽快清醒，想要使闹钟正确地停止，使用者必须计算一道简单的数学题（三个1~100的数字相加，数字在到达闹钟预设时间后随机给出），并按动第一个按钮对应的次数，使屏幕上显示的等式成立（屏幕上显示的等式为“x+y+z=count”，每按动一次按钮使count值加一，count初始值为零；x、y、z均为随机数）。若不小心按过头，只能按下第二个按钮重置count值到零。
如此一来，相信使用者能够很快清醒过来，更妙的是————使用者可以通过大力按动按钮来发泄自己的起床气！天呐真是太精妙的设计！！！！

为了使使用者尽快清醒，想要使闹钟正确地停止，使用者必须计算一道简单的数学题（三个1~100的数字相加，数字在到达闹钟预设时间后随机给出），并按动蓝色按钮对应的次数，使屏幕上显示的等式成立（屏幕上显示的等式为“x+y+z=count”，每按动一次按钮使count值加一，count初始值为零；x、y、z均为随机数）。若不小心按过头，只能同时按下蓝色和绿色按钮重置count值到零。
如此一来，相信使用者能够很快清醒过来，更妙的是————使用者可以通过大力按动按钮来发泄自己的起床气！天呐真是太精妙的设计！！！！
强制唤醒功能
若使用者很不幸，在闹钟有效振铃时间内（若不对闹钟做任何事，它将会振铃五分钟后自动停止）仍未清醒，或使用某种能够穿越时空的魔法免疫了闹钟，或闹钟没有将使用者叫醒而是将他的大冤种室友叫醒，则室友可以使用这个功能来将使用者强制唤醒。同时按住所有按钮（黄、绿、蓝色），闹钟将发出一种更大声、更强力的声音，以便将其唤醒。

Arduino

Arduino是一款便捷灵活、方便上手的开源电子原型平台。包含硬件（各种型号的Arduino板）和软件（ArduinoIDE）。由一个欧洲开发团队于2005年冬季开发，其完全开源，有许多经常维护的库可以直接调用，对于新手或是业余爱好者来说是非常简易的平台。事实上，Arduino开发板就是加强版的单片机，功能更强大，使用更简单。需要指出的是，如果您是相关专业或是想要长期从事这项爱好的读者，其实并不建议深入地学习Arduino，而应该去学习单片机。如果您单片机学好了，Arduino只要一下午就可以达到精通。
上文已经提及，Arduino由各种型号的开发板和软件（ArduinoIDE）组成。

Arduino IDE

在ArduinoIDE中编写好程序，然后上传到开发板中，开发板就可以完成一系列您想要做的事。ArduinoIDE可以在官网直接下载，这里不做更多说明了。打开ArduinoIDE后，您可以看到已经为您准备好了一些代码：具有很少的英文知识的读者就完全可以看懂。做一些注释如下： 
void setup() {
  // put your setup code here, to run once:      //把你用来设置的代码放在这里，它们将只运行一次：


}


void loop() {
  // put your main code here, to run repeatedly:        //把你的主体代码放在这里，它们将被重复执行：


}

Arduino板

我在这次星火杯中使用的是Arduino UNO板。这是很适合新手使用的板子。Arduino板分为官方板和一种通常称作“行家改进板”两种，通俗的讲，因为Arduino是完全开源的，任何人只要有意愿都可以生产Arduino板。官方板就是字面意思，因为是进口的，所以价格会贵上不少。有的视频建议初学者使用官方板开始学习，因为质量有保证，以防出现板子出现问题而不知道如何解决的情况，但在我本人使用的过程中，自己感觉两者其实没什么区别。

面包板

面包板是搭建电路时常用的器材。使用时，将各种线插在面包板的合适位置，能够大大简化排线难度，使作品更美观，也更不容易出错。如果你从前没有接触过面包板，此时手里拿着一个面包版的话应该是不知所措的。常见的面包板后面是有一层胶粘着的，阻碍了我们的观察。我已经把我的面包板剥开了，你们不用剥自己的。 如图所示，可以清晰地看出一块面包版由三块儿组成。其中有两块小的是完全贯通的。在正面，我们也可以看到上面表示的“+”和“-”的字样。它们可以被称作电源轨。arduino板一般只有一个5V口，而将arduino板的正负口通过跳线连接在面包板上的位置，就可以使整个电源轨都具有正负性。这样，再连接别的器件需要正负极时，只需要连在电源轨上即可。大的那部分面包板，又可以看到上下两部分中是分别连通的。事实上，那是五个孔为一组的。我们应该对面包板的内部构造了如指掌，才能在使用的时候不会出错。
