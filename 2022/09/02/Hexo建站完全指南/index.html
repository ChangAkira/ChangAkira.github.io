安装git

Git 是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。 Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。 Git 与常用的版本控制工具 CVS, Subversion 等不同，它采用了分布式版本库的方式，不必服务器端软件支持。

（选做）（慎选）

当我们在执行npm install express -g命令时，g表示global全局。会默认下载到c盘，如果想要改变这令人头大的怪现状的话，那可以————改路径！

在nodejs安装目录下创建两个文件夹，名称分别为node_global和node_cache，在node_global文件夹下再建一个node_modules文件夹。 win+R打开运行窗口，打开cmd，再输入以下两条指令：

npm config set prefix "创建的node_global文件夹所在路径"
npm config set cache "创建的node_cache文件夹所在路径"
打开计算机属性，选择高级系统设置。打开环境变量

在系统变量中，新建变量名为：NODE_PATH 变量值：node_global文件夹下的node_modules文件夹。
修改用户变量中的Path变量，将默认的npm路径修改为新建的node_global路径。

安装Hexo

新建一个文件夹（如blog）。在文件夹中右键，“Git bash here”。输入命令：npm install -g hexo-cli来安装Hexo。安装完成后可以分别使用命令：git -vgit -v
